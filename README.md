# Task Manager (C++)

This is a simple task manager built using C++.

The main idea of this project is to practice object-oriented programming
and understand how linked lists work in a real example.

## What it does

- Add tasks
- Show all tasks
- Delete a task by ID

## Concepts used

- Classes and objects
- Pointers
- Linked list (dynamic memory)
- Basic input/output

## How to run

Compile:
g++ main.cpp TaskManager.cpp -o app

Run:
./app

## Notes

The code is written in a simple way for learning purposes.

This project can be improved later by:
- Saving tasks to a file
- Adding update/edit feature
- Using other data structures like stack or queue
